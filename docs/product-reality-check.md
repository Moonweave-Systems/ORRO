# ORRO Product Reality Check

Product Reality Check is a local dogfood checklist. It is not proof,
verification, telemetry, analytics, data collection, approval, assurance, or a
benchmark claim.

Questions:

- Did ORRO reduce steps?
- Did ORRO avoid unnecessary tool/model calls?
- Did ORRO make next action clear?
- Did ORRO reduce reviewer burden?
- Did ORRO block missing proofcheck/handoff gates?
- Did ORRO stop instead of guessing?
- Did artifact count confuse the user?
- Did report reduce artifact fatigue?

ORRO should recommend the smallest safe workflow, avoid wasteful AI/tool work,
and stop rather than overclaim.

```text
Depone verifies; witnessd executes; ORRO exposes the workflow.
```
